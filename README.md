# Enterprise Slide Show

## Install ##

From your terminal run:

```bash
    git clone git@github.com:Hijolt/redux-starter.git 
```

Then once it's downloaded

```bash
    npm install
```

## Run the dev server ##

From your terminal run:

```bash
    npm run start 
```

Or in cloud 9:

```bash
    webpack-dev-server  --host $IP --port $PORT  --hot --inline
```
